宣传自家节目

《Web Worker》播客

自家节目：上周发布了 web worker 播客 No.4 邀请到了 HR 做嘉宾！我要求面试涨薪。

还挺火，上了两次小宇宙新星榜，公众号文章已经 1300+ 阅读量了。对一个刚开始做的节目来说是一个莫大的鼓励。

🔧 技巧 #

  • edge 浏览器的断网小游戏 网址 edge://surf/ - 大家都知道 Chrome 中如果无网环境会出现小恐龙游戏，没想到 edge 有一个更炫酷的网上冲浪游戏。输入 edge://surf/ 玩起来吧。

  • 忘了 xshell 吧这个更好：VSCode Remote Development - 辛宝 -给后端老哥演示了一下用 vscode remote 直接操作测试服务器，老哥们都惊掉了下巴，建议大家反复阅读，掌握这项技能更容易操作服务器。

🔗 工具和链接 #

  • 山月大佬的：前端这些有趣的库 -山月大佬整理了很多工具库，我看了大为震撼，日积月累积少成多。

  • 神器 BundleScanner：扫描网站中的 js 匹配工具类库 - 打开 BundleScanner 输入网址，就能扫描匹配用到的工具类库，具体到小版本，这可比 wappalyzer 强大多了。以及 作者的思考。

💡 共读 #

  • 为什么 Vue 在国际上越来越没影响力？ - 尤雨溪的回答 - 作者亲自回复，答案是否定的。
    • 从 npm 下载量和 chrome 插件角度看持续上涨
    • vue 的市场定位和应用场景的渗透

  • Modern.js 每周更新：支持 Windows、初步提供部署功能 - Modern.js - 文章以极大的耐心说明了 Modern.js 本周的更新，除了标题描述细致，还补充了操作视频，补充了修改说明。我认为是开源 changelog weekly 的天花板了，持续关注。

  • Web Infra 大咖面对面：赵海平与张宏波聊聊编程语言 - web infra 大佬们聊。听下来感受到大佬更高层次维度的思考，值得一听。比如 39'00 谈 rust 是 js 基础的未来。看主持人小李如何 1v2 进行互动，我也参考学习。
